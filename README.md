# Doors 7

Doors 7 é uma distribuição Linux em desenvolvimento inspirada na simplicidade, leveza e experiência de uso do Windows 7.

## Objetivos

- Interface inspirada no Windows 7
- Desempenho leve
- Código aberto
- Compatibilidade com aplicativos Windows por meio de ferramentas como Wine
- Respeito à privacidade do usuário
- Gratuito

## Estado do projeto

🚧 Em desenvolvimento inicial.

Atualmente o projeto está na fase de pesquisa e planejamento.
